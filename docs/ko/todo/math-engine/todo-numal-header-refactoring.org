* 11개 Header

| Header | Math Engine 역할 | 핵심 판단 |
|--------+------------------+-----------|
| =numal.h= | aggregate | 승인 component inventory |
| =scalar.h= | scalar | generic macro, pi/angle, tolerance와 checked result |
| =dualnumber.h= | automatic differentiation value | layout와 derivative dimension |
| =vec3.h= | vector value | tolerance, aliasing, buffer/string 분리 |
| =quat.h= | rotation value | enum 고정, normalization와 frame |
| =dualquat.h= | rigid transform value | tag/layout와 output order |
| =plane.h= | primitive value | normal/tolerance/unit와 failure |
| =geom.h= | geometry compatibility | canonical geometry API로 forwarding |
| =vec3.hpp= | C++ façade | namespace/source compatibility 결정 |
| =quat.hpp= | C++ façade | exception/compiler ABI 결정 |
| =dualquat.hpp= | C++ façade | package/internal 분리 판단 |

Header별 함수 수준 리팩터링은 별도 manifest에서 declaration, definition, root export,
install, wrapper와 test를 연결한다. 기존
[[file:../todo-리팩토링-foundation-headers.org][Foundation Header TODO]]는 foundation ABI
gate만 소유하고 Math Engine API 설계는 이 문서가 소유한다.
