layout, 함수 이름과 source를 BYUL public ABI로 자동 복사하지 않는다. GPU tool header
리팩터링과 분리해 Math Engine =mathunits= migration만 이 문서가 소유한다.

29개 initial asset은 단일 ownership manifest에 각각 기록한다. 기존 source/header를
canonical Mathunits 구현으로 복사하지 않고 requirement, data, regression, compatibility
또는 archive 중 하나의 disposition을 가진다.

* Parent Mathunits 합류 Gate

이 문서는 [[file:todo-mathunits.org][Mathunits 계획]]의 migration work package다.

- TODO 1~2는 Mathunits TODO 1의 provenance/requirement 증거다.
- TODO 3은 Mathunits TODO 2가 승인한 ABI 뒤에 실행한다.
- TODO 4는 별도 public API를 구현하지 않고 Mathunits TODO 3의 canonical 구현에
  migration corpus와 generated data를 합류한다.
- 기존 =tools/unit_1003= compatibility/archive 종료는 전체 Header 리팩터링 8단계
  Tool/reference 계획이 소유한다.
