* 목적

Scalar 초월함수, 특수함수와 확률분포 값을 =numal= 위의 독립 =mathfunc= 계층으로
제공한다. Parser, RNG sampling, GUI와 Balix state에는 의존하지 않는다.

* 범위

- exp/log, trigonometric와 hyperbolic 함수
- gamma/beta/erf/Bessel/Lambert W 후보
- complex 함수와 branch metadata
- PDF/PMF/CDF/inverse CDF의 순수 값 계산
- f32/f64와 지원 가능한 고정밀 backend
- domain, pole, overflow, underflow, inexact와 estimated error
