* 목적과 단일 소유권

=byul/rng=의 deterministic stream, uniform value, distribution sampling, fill, roll과
shuffle을 Math Engine RNG/sampling component로 통합한다. Foundation과 simulation
consumer는 seed/stream을 주입하고 결과를 사용할 뿐 random algorithm이나 distribution
공식을 다시 구현하지 않는다.

확률분포의 PDF/PMF/CDF/inverse CDF 같은 순수 값 함수는
[[file:todo-mathfunc.org][Mathfunc]]가 소유하고, RNG state를 소비해 sample을 만드는
알고리즘은 이 문서가 소유한다. 게임 loot, 보상과 재시도 정책은 consumer 책임이다.

* 기존 Header 6개

| Header | Math Engine disposition |
|--------+-------------------------|
| =byul/rng/rng.h= | canonical module entry·version |
| =byul/rng/rng_core.h= | deterministic engine state·stream·uniform primitive |
| =byul/rng/rng_fill.h= | typed buffer fill |
| =byul/rng/distributions.h= | distribution sampling; pure probability 값은 Mathfunc 연결 |
| =byul/rng/roll.h= | discrete·weighted sampling |
| =byul/rng/shuffle.h= | generic/typed permutation |

* 계약 경계

- 기본 generator와 stream mapping은 algorithm/version을 고정해 deterministic replay를
  보장한다.
- Seed, stream, copy, serialization, thread sharing과 reentrancy 계약을 명시한다.
- Range bias, invalid bound, non-finite parameter와 empty population을 정상 sample과
  구분한다.
- Sampling 함수는 caller-owned RNG state만 변경하며 global hidden state를 사용하지 않는다.
- 암호학적 난수 보장은 별도 backend/contract 없이는 주장하지 않는다.
