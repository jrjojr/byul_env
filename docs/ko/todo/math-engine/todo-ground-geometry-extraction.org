* 목적과 경계

=byul/ground/ground.h=와 source의 heightfield sample, normal과 raycast 계산을 Math Engine
terrain/geometry query로 옮긴다. Ground는 tile/hash/height buffer, mapper callback,
material과 lifecycle을 소유하고 neutral view를 Math Engine query에 전달한다.

| Math Engine | Ground consumer |
|-------------+-----------------|
| heightfield interpolation·normal | height/table/hash storage |
| ray/terrain intersection query | tile mapper callback과 material lookup |
| caller-owned terrain view·query result | init/assign/reset/free와 allocation |
| unit/frame/tolerance/status | world policy와 result 적용 |

Canonical Math Engine header는 =ground_t=나 Ground callback을 받지 않는다. Ground
adapter에는 interpolation/raycast 공식을 복제하지 않는다.

* Canonical/Consumer Gate
