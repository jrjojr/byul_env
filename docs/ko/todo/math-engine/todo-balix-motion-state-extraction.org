* Public value surface

=motion_state.h=는 다음 세 public value layout을 노출한다.

| Public type | Field | ABI/목표 책임 |
|-------------+-------+---------------|
| =linear_state_t= | =position=, =velocity=, =acceleration= | 기존 Balix ABI와 lifecycle은 유지하고 adapter가 neutral kinematics value로 변환한다. |
| =attitude_state_t= | =orientation=, =angular_velocity=, =angular_acceleration= | 기존 Balix ABI와 lifecycle은 유지하고 unit/frame을 명시해 neutral pose/twist로 변환한다. |
| =motion_state_t= | =linear=, =angular= | Balix aggregate DTO이며 Math Engine public header에 직접 노출하지 않는다. |

Init/full/assign 함수는 계산 kernel이 아니다. 다만 Math Engine의 neutral value type에도
zero/identity constructor가 필요하면 별도 canonical value API가 소유하며, 기존
=*_state_*= symbol이 그 type으로 소유권을 옮긴다는 뜻은 아니다.

* 선언·구현과 검증 공백

- =motion_state.cpp=에는 9개 lifecycle definition만 있고
  =motion_state_calc_collision_time=과 =motion_state_calc_collision_point= definition이 없다.
- 두 collision 선언은 =radius_a=, =radius_b=, =max_t=와 =bool=만 제공하며 negative
  radius, non-finite, initial overlap, tangent, acceleration model, frame과 tolerance 계약이
  없다.
- time과 point를 별도 함수로 계산하면 서로 다른 root/tolerance가 선택될 위험이 있으므로
  canonical API는 =status + toi + point + normal/feature + residual=의 단일 result다.
- 현재 =byul/balix/tests/CMakeLists.txt=에는 motion-state 전용 test source가 없다. 미구현
  두 export를 link하는 ABI fixture도 없다.

* Build·DLL 이행

현재 =motion_state.cpp= 전체가 =balix_SOURCES=와 root shared source inventory에 포함된다.

- Static 과도기: 9개 lifecycle definition은 Balix source에 남긴다. 새 neutral value와
  sphere CCD kernel은 단일 Math Engine manifest가 소유하고 Balix가 static target을
  link한다. 누락된 2개 definition은 Balix legacy forwarding shim으로만 추가한다.
- 독립 DLL 전환: 중앙 규칙과 =rules-ref.org= revision 승인 뒤 Math Engine DLL이
  canonical CCD symbol을 export한다. Root BYUL은 기존 두 이름의 shim을 export하되
  Math Engine 계산 source를 중복 compile하지 않는다.

* 경계

Math Engine은 neutral position/velocity/acceleration와 pose/twist POD 또는 array view를
소유한다. Balix =motion_state_t=의 field layout을 Math Engine public ABI로 복사하지
않고 explicit converter를 둔다. Collision time/point는 하나의 CCD result에서 함께
반환해 두 번 계산하거나 결과가 불일치하지 않게 한다.
