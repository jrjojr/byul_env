* 목적

Compiled expression 또는 direct callback에서 renderer 독립 graph sample, root,
교점, 극값과 불연속 후보를 계산한다. Display와 viewport interaction은 application
책임이며 수치 미분·적분·보간은 =numeq=를 재사용한다.

* 범위

- Cartesian, parametric, polar와 implicit scalar field sampling
- adaptive subdivision과 error estimate
- root/intersection/minimum/maximum/inflection 후보
- discontinuity와 asymptote 구간 분리
- Marching Squares용 cell/segment data
- progressive refinement, cancellation, budget와 deterministic cache key
