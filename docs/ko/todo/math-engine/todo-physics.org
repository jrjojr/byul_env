* 책임

Math Engine =physics= 계층은 외부 state를 소유하지 않는 물리 법칙과 model을 제공한다.
Balix는 body/environment/world state를 neutral parameter로 변환하고 결과를 적용한다.

| 영역 | 공식 | 단위·주의 |
|------+-------+-----------|
| kinematics | =p=p0+v0*t+0.5*a*t^2=, =v=v0+a*t= | m, s, m/s, m/s^2 |
| drag | =F=-0.5*rho*Cd*A*|v|v= | kg/m^3, m^2, kg; relative air velocity |
| energy | =Delta KE= | joule, dissipated energy는 음수 금지 |
| contact | normal/tangent 분해와 restitution | normal은 unit 또는 명시 normalization |
| spin | Magnus/gyro 후보식 | coefficient 차원과 적용 범위를 명시 |
| angular drag | torque/inertia 기반 model | mass를 inertia로 대신하지 않음 |
| signal | sinusoidal periodic field | rad/s와 Hz 변환 명시 |

보편 상수 =g0=, 표준 대기 밀도와 압력은 =mathunits/constants=가 출처, 단위,
불확도와 data version을 소유한다. =Cd=.47=, cross-section, restitution, friction,
Magnus/gyro gain과 MPC weight는 보편 상수가 아니라 Balix 또는 product preset이다.

Fundamental constant의 외부 기준은
[[https://physics.nist.gov/cuu/Constants/index.html][NIST/CODATA database]]의 edition, 값, 단위와
uncertainty 계약을 사용한다. 표준 대기나 재료·aerodynamic coefficient는 CODATA
fundamental constant가 아니므로 각각의 별도 표준/모델 provenance가 필요하다.

현재 atmosphere의 humidity/temperature/pressure 계수와 spin drag scale은 근거가
불분명한 heuristic이다. 이름을 물리 상수처럼 유지하지 않고 versioned empirical model
또는 tuning function으로 분류하고 유효 범위 밖을 명시한다.

* Canonical/Consumer Handoff

TODO 1~3은 Header 리팩터링 5단계의 canonical Physics work다. TODO 4의 DTO mapping,
legacy forwarding과 lifecycle은 [[file:todo-balix-math-engine-consumer.org][Balix
consumer 계획]]과 전체 Header 리팩터링 Simulation consumer 단계가 실행하며 이
문서는 결과만 합류한다. Canonical Physics handoff는 TODO 1~3과 static conformance로
판정한다.
