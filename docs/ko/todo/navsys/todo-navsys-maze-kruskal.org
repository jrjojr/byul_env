* 구현 상세 감사

| 부분 | 현재 | 결함/개선 후보 |
|------+-----+----------------|
| vertices | odd/odd cell을 passage로 설정 | dimension checked지만 allocation upper bound 없음 |
| edges | east/south adjacent wall을 한번씩 추가 | edge set은 undirected grid의 모든 edge를 정확히 한번 포함 |
| randomization | entire edges =shuffle= | explicit seed 없음; shuffle 후 분포는 random edge order이나 UST uniform 아님 |
| DSU | 2D parent, recursive find path compression | rank/size union 없음, recursion/2D storage 불필요; parent init은 cell positions만 유효 |
| loop | 모든 edge를 끝까지 처리 | component가 1이면 조기 종료 가능; cancellation/step stats 없음 |
| allocation | maze/vector/parent/walls/hash | C++ exception barrier와 hash insert failure 없음 |
| test | non-NULL과 ASCII | exact =V-1=, connectivity/cycle/DSU invariant 없음 |

* 목표 계약

- algorithm 이름에 randomized를 명시하며 minimum total weight/UST guarantee를 주장하지 않는다.
- DSU는 logical vertex id 0..V-1, union-by-rank/size + path compression을 사용한다.
- accepted edge가 =V-1=이면 종료하고 component count와 considered/accepted stats를 기록한다.
- seed/RNG/version, cancellation/edge-consideration limit과 output strong guarantee를 제공한다.
