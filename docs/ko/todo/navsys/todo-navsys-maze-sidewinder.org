* 대상과 P0 판정

- 대상: =maze_sidewinder.h/.cpp=.
- 공통 API는 [[file:todo-navsys-industry-api-baseline.org][Navsys 업계 API 기준]]을 따른다.
- [[https://weblog.jamisbuck.org/2011/2/3/maze-generation-sidewinder-algorithm.html][Jamis Buck reference]]는 first row를 끝까지 하나의 corridor로 만들고, 이후 각
  run은 끝날 때 이전 row로 정확히 하나 이상 연결한다.
- 현재 구현은 첫 logical row에서도 random하게 run을 닫지만 north carve를 하지 않는다.
  따라서 first row가 여러 disconnected run으로 끊길 수 있는 P0 correctness 후보이다.

* 목표 계약

- selected sweep 방향과 이전-row 방향을 explicit option으로 나타내며 legacy는 east/north를 유지한다.
- boundary-adjacent first row는 sweep 방향으로 모두 연결한다. 이후 모든 run은 끝날 때 이전 row로
  정확히 하나 연결한다.
- 결과는 perfect maze이나 UST가 아니고 sweep 방향 bias 및 이전-row 방향 dead-end 특성을 가진다.
- seed/RNG draw order, close probability(기본 1/2), cancel/max cells와 status를 명시한다.
