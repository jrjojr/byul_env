* 대상과 알고리즘 판정

- 대상: =maze_aldous_broder.h/.cpp= 및 maze module tests.
- 현재 구현은 odd interior cell graph에서 simple random walk를 수행하고 처음 방문한 edge만
  carve한다. 유한 connected rectangular grid라면 uniform spanning tree를 생성한다.
- [[https://epubs.siam.org/doi/abs/10.1137/0403039][Aldous의 원 논문]]과
  [[https://www.boost.org/doc/libs/1_48_0/libs/graph/doc/random_spanning_tree.html][Boost random spanning tree API]]를 정확성/API 기준으로 사용한다.
- 공통 status/options/cancellation은
  [[file:todo-navsys-industry-api-baseline.org][Navsys 업계 API 기준]]을 따르며 아래 signature보다 우선한다.

* 책임 경계와 실제 검증 파일

- 이 문서는 Aldous-Broder algorithm state·random walk·materialization과 RNG 소비,
  seed/replay, budget/cancel 구현을 소유한다.

* 구현 단위 감사

| 단계 | 현재 구현 | 세부 결함/결정 |
|------+-----------+----------------|
| validation | width/height odd >=3 | signed overflow와 최대 allocation 크기 없음 |
| allocation | =maze_create_full= 직후 dereference | maze/hash 실패 검사 없음 |
| graph | =(1,1)..(w-2,h-2)= step 2, 4-neighbor | 이 topology를 public contract로 고정해야 uniform claim이 유효 |
| start | 모든 interior odd cell vector 후 uniform pick | 후보 count overflow와 empty 방어를 status로 변환 |
| RNG | =mt19937(time(nullptr))= | 초 단위 seed collision, caller seed 불가, engine/version 기록 불가 |
| walk | 매 step directions shuffle 후 첫 valid neighbor | valid neighbor에 균등하지만 shuffle 비용; direct uniform index 가능 |
| termination | visited count가 전체 cell 수가 될 때 | 이론상 almost-sure 종료이나 deterministic upper bound 없음 |
| result | non-passage를 blocked hash에 insert | insert 실패 무시해 incomplete/잘못 열린 maze 반환 |
| test | block count range와 print | tree/connectivity/uniform/reproducibility/termination 미검증 |

* 목표 계약

- 입력 topology는 odd >=3의 rectangular 4-neighbor cell graph이며 외곽은 wall이다.
- 결과 passage graph는 =V-1= carved edge, connected, acyclic인 perfect maze다.
- 모든 spanning tree가 동일 확률이라는 claim은 동일 graph에서 unbiased neighbor choice를 사용할
  때만 허용한다. max-step로 중단된 결과는 반환하지 않는다.
- seed와 RNG algorithm/version을 result provenance에 기록한다. 같은 seed 재현 범위는 명시된
  engine/version/platform matrix에 한정한다.
- cancel은 walk와 blocked materialization 양쪽에서 bounded interval로 poll한다.
