* 대상과 opaque resource 판정

- 대상: =obstacle_core.h/.cpp=, =obstacle_t= lifecycle/mutation/neighborhood/navgrid bridge.
- owning =coord_hash_t*=가 public layout에 있으므로 목표는 opaque handle이다. ABI 1.x additive
  accessor/status/buffer/overlay 이행 후 ABI-major에서 struct definition을 숨긴다.

  [[file:todo-navsys-industry-api-baseline.org][Navsys 업계 API 기준]]이 우선한다.

* 현행 구현 결함

| 영역 | 현재 | 영향 |
|------+-----+------|
| create | malloc 후 hash create 실패 미검사 | half-initialized resource |
| copy | 임시 hash destroy 후 copy, failure 미검사 | crash/leak/invalid copy 가능 |
| origin | fields만 변경 | blocked world coordinates와 metadata 불일치 |
| resize | fields만 변경 | blocked coordinates outside extent 잔존 |
| block | heap =coord_create= 후 hash replace result 무시, 항상 true | allocation/hash failure 은폐, 불필요 heap |
| unblock | hash remove result 반환 | 비교적 관찰 가능하지만 invalid/outside policy 없음 |
| range | square =(2r+1)^2= | Euclidean radius처럼 읽힐 수 있고 overflow/cancel 없음 |
| straight | obs NULL/negative range validation 없음 | NULL crash 가능, negative가 radius 0으로 silently 처리 |
| neighbor allocation | list/hash/coord allocation failure 무시 | partial collection 성공 |
| raw hash | borrowed internal pointer | lifetime/invalidation/thread 누출 |
| apply/remove | void, every cell block/normal | partial/OOM 숨김; 다른 source terrain/obstacle를 remove가 지움 |
| iterator | create failure 미검사 | NULL iterator use 가능 |
| export | =obstacle_is_coord_blocked= no =BYUL_API= | installed header가 외부 사용을 유도하나 Windows root export 불가 가능 |
| test | shape 중심, core copy/origin/resize/apply overlap 부족 | resource invariants 미보호 |

* 업계 비교와 target API

- [[https://www.sqlite.org/c3ref/sqlite3.html][SQLite opaque handle]]처럼 public resource layout을 숨긴다.
- [[https://recastnav.com/classdtNavMeshQuery][Detour]]와 공통 baseline처럼 raw container 대신 capacity/count/status를 사용한다.
- [[https://gdal.org/en/stable/api/gdal_alg.html][GDAL rasterization]]처럼 line/polygon policy는 generator layer가 소유하고 core는 coordinate
  set lifecycle/mutation만 담당한다. neighbor/degree helpers는 navgrid/shared geometry로 이동 후보이다.
