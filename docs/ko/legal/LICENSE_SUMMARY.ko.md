# BYUL 라이선스 요약

이 문서는 BYUL 라이선스를 쉽게 이해하기 위한 요약입니다.

법적으로 우선하는 문서는 루트의 [`LICENSE`](../../LICENSE) 파일입니다. 이 요약과 원문 라이선스 내용이 충돌하면 루트의 `LICENSE`가 우선합니다.

## 라이선스 종류

BYUL은 다음 라이선스로 배포됩니다.

```text
Byul World Source-Available Non-Commercial License v1.0
```

이 저장소는 **오픈소스가 아니라 source-available 프로젝트**입니다.

코드를 읽고, 학습하고, 빌드하고, 제한된 비상업적 조건에서 평가할 수는 있지만, 자유롭게 재배포하거나 상업적으로 사용할 수는 없습니다.

## 허용되는 것

다음은 허용됩니다.

- 소스 코드를 읽고 학습하는 것
- 개인 학습 목적으로 빌드하고 실행하는 것
- 학술 연구 또는 기술 검토를 위해 분석하는 것
- 예제나 데모를 비상업적 평가 목적으로 사용하는 것
- 개인 또는 학술 목적의 로컬 비공개 수정본을 만드는 것
- 저장소를 인용하고 비상업적 벤치마크 결과나 분석을 공개하는 것
- 개인 학습, 이슈 보고, 기여 논의, 비상업적 기술 검토 목적의 GitHub fork

## 서면 허가 없이 금지되는 것

다음은 사전 서면 허가 없이는 금지됩니다.

- BYUL을 상업적으로 사용하는 것
- 유료 제품, 유료 서비스, 고객 작업, 상업 게임, 상업 시뮬레이션, 회사 내부 업무 시스템에 사용하는 것
- 소프트웨어를 재배포, 재게시, 미러링, 재호스팅, 판매, 재라이선스, 패키징하는 것
- 수정본, 바이너리, SDK, 라이브러리, 플러그인, 파생 패키지를 배포하는 것
- BYUL을 라이브러리, SDK, 플러그인, 서비스, 의존성으로 제공하는 것
- 라이선스, 저작권, 출처 표시를 제거하는 것
- BYUL의 저작자나 소유권을 허위로 주장하는 것
- BYUL 관련 이름을 공식 승인이나 제휴처럼 보이게 사용하는 것

## GitHub Fork

GitHub fork는 제한된 비상업적 목적에 한해 허용됩니다.

- 개인 학습
- 이슈 보고
- 기여 논의
- 기술 검토

fork를 공개 미러, 수정 릴리스, 패키지 배포, 상업적 의존성, 이름만 바꾼 프로젝트로 사용하면 안 됩니다.

## 상업 라이선스

상업적 사용은 별도 서면 허가가 필요합니다.

상업 협업, 유료 프로젝트 통합, 사업 평가, 상업 게임 사용, 시뮬레이션 사용, SDK/라이브러리 패키징, 별도 라이선스 문의는 아래로 연락하십시오.

```text
byuldev@outlook.kr
```

저작자는 이 소프트웨어로 인해 발생하는 손해, 손실, 청구에 대해 책임지지 않습니다.
